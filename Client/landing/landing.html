Giới thiệu Đồ án & Team

Dự án tập trung vào các tính năng: giám sát màn hình, điều khiển webcam, ghi âm, quản lý tiến trình, quản lý file, và terminal logs — tất cả được kết nối qua socket với khả năng mở rộng linh hoạt.

  • Giao diện Soft UI đồng bộ
  • Tối ưu cho Dark/Light mode
  • Trải nghiệm mượt, responsive

Remote Control System Overview

Soft shadows • Smooth interactions • Clean tokens

Tính năng cốt lõi

Các module kỹ thuật chính của hệ thống.

Quản lý Ứng dụng

Điều phối app với giao diện Soft UI, cập nhật trạng thái thời gian thực.

Quản lý Tiến trình

Theo dõi CPU/RAM và thao tác process qua socket hai chiều.

Chụp màn hình

Truyền tải khung hình nén thời gian thực qua Socket.

Keylogger

Ghi nhận sự kiện phím với buffer an toàn, stream về dashboard.

Điều khiển Nguồn

Gửi lệnh shutdown/restart qua kênh bảo mật, xác nhận hai bước.

Giám sát Webcam

Streaming video/audio với kiểm soát start/stop và bitrate.

Chức năng sáng tạo

Các tính năng nâng cao được phát triển để tăng điểm thưởng cho đồ án.

Quản lý File từ xa

Duyệt cây thư mục, tải xuống tập tin từ máy bị điều khiển thông qua FileManagerService.cs. Hỗ trợ đầy đủ các thao tác: duyệt, tải xuống, upload, đổi tên, xóa file và thư mục.

Giám sát tài nguyên hệ thống

Theo dõi biểu đồ thời gian thực của CPU và RAM bằng các widget chuyên biệt. Hiển thị thông tin phần cứng chi tiết và thống kê hiệu suất liên tục.
