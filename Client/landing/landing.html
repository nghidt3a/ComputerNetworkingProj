N-SIGHT: Giải Pháp Quản Trị Hệ Thống Từ Xa Toàn Diện

Hệ thống điều khiển máy tính thời gian thực với độ trễ thấp, bảo mật cao và giao diện Soft UI hiện đại. Giải pháp tối ưu cho việc giám sát, hỗ trợ kỹ thuật và quản lý tài nguyên tập trung.

Giới thiệu Dự án

N-SIGHT là dự án phần mềm điều khiển từ xa (Remote Desktop Control) được phát triển bởi nhóm sinh viên lớp 24CTT5 - ĐH KHTN. Hệ thống vượt qua các giới hạn của các ứng dụng quản lý truyền thống bằng cách tích hợp khả năng giám sát đa chiều: hình ảnh, âm thanh, tệp tin và tiến trình hệ thống, tất cả trên nền tảng Web hiện đại.

  • Real-time Streaming: Truyền tải hình ảnh/âm thanh độ trễ thấp qua WebSocket.
  • Giao diện Soft UI: Thiết kế chuẩn mực, thân thiện và hỗ trợ Dark Mode.
  • Đa nền tảng: Client hoạt động trên mọi trình duyệt hiện đại.

Kiến Trúc & Hiệu Năng

Sử dụng công nghệ đa luồng (Multithreading) và giao thức TCP/IP tối ưu hóa băng thông.

Tính Năng Cốt Lõi

Các module chức năng chính đảm bảo khả năng kiểm soát toàn diện.

Quản Lý Ứng Dụng

Khởi chạy và tìm kiếm ứng dụng trên máy Target nhanh chóng từ thư viện tích hợp.

Kiểm Soát Tiến Trình

Giám sát tài nguyên (PID, RAM) và can thiệp (Kill/Start) vào các tiến trình hệ thống.

Remote Desktop

Stream màn hình thời gian thực chất lượng cao, hỗ trợ thao tác chuột và phím ảo.

Keylogger Thông Minh

Ghi nhận và đồng bộ sự kiện bàn phím thời gian thực về Dashboard.

Điều Khiển Nguồn

Thực hiện lệnh Shutdown, Restart, Sleep, Lock màn hình từ xa.

Giám Sát Webcam

Truy cập Camera/Microphone, hỗ trợ ghi âm và quay video giám sát.

Chức Năng Nâng Cao

Các tính năng mở rộng kỹ thuật cao (Advanced Requirements).

File Manager Đa Nhiệm

Hệ thống quản lý tệp tin toàn diện: Duyệt cây thư mục, thao tác Upload/Download tốc độ cao, Đổi tên, Xóa file/folder. Giao tiếp dữ liệu dạng Binary/Base64 an toàn.

Biểu Đồ Hiệu Năng Real-time

Dashboard trực quan hóa dữ liệu phần cứng (CPU, RAM, Disk, GPU) thông qua thư viện Chart.js. Cập nhật liên tục mỗi giây để phát hiện điểm nghẽn hệ thống.
