Giới thiệu Đồ Án

N-SIGHT là đồ án môn học Mạng máy tính, được thực hiện bởi Nhóm 17 — Lớp 24CTT5, Đại học Khoa học Tự nhiên, ĐHQG-HCM.

Đồ án xây dựng ứng dụng điều khiển máy tính từ xa với các chức năng như xem màn hình, điều khiển chuột/bàn phím, giám sát webcam, thu âm, quản lý tiến trình và tệp tin. Hệ thống sử dụng giao thức WebSocket để truyền dữ liệu thời gian thực giữa Client và Server.

  • Real-time Streaming: Truyền hình ảnh và âm thanh qua WebSocket
  • Giao diện Web: Dashboard thân thiện, hỗ trợ chế độ tối
  • Đa nền tảng: Chạy trên mọi trình duyệt hiện đại

Kiến Trúc Hệ Thống

Web Client (Controller)

HTML/CSS/JS — Giao diện điều khiển

WebSocket Protocol ws://ip:8181 — JSON + Binary Stream

C# Server (Target Machine)

Command Router, Screen Capture, Audio/Video Stream

Hệ điều hành bị điều khiển

Process, FileSystem, Input/Output, Hardware

Communication Flow

Client gửi lệnh JSON qua WebSocket, Server phản hồi dữ liệu JSON hoặc binary stream

Real-time Streaming

Server capture màn hình/webcam, encode thành binary và stream liên tục tới Client

Authentication

Client xác thực bằng password khi kết nối, Server kiểm tra trước khi cho phép điều khiển
